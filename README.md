# 🚀 TradeSignal Pro

**Application d'analyse technique gratuite avec signaux IA en temps réel**

## ✨ Fonctionnalités

- ✅ **Interface moderne** - Design glassmorphism professionnel
- ✅ **12 Timeframes** - De 1 seconde à 1 heure (incluant 3s)
- ✅ **Signaux ultra-précis** - Algorithmes IA (RSI, MACD, EMA, Volume)
- ✅ **Alertes sonores** - Notifications audio pour les signaux BUY/SELL
- ✅ **Auto-refresh** - Mise à jour temps réel configurable
- ✅ **100% Gratuit** - Hébergement gratuit sur GitHub Pages
- ✅ **Responsive** - Mobile et Desktop

## 🎯 Indicateurs Techniques

- **RSI (14)** - Détection des zones de surachat/survente
- **MACD** - Analyse du momentum et des tendances
- **EMA 9/21** - Moyennes mobiles exponentielles
- **Volume** - Analyse des flux de volume
- **Volatilité** - Mesure de la volatilité en temps réel

## 🚀 Déploiement sur GitHub Pages

### 1. Créer un repository GitHub

```bash
# Créer un nouveau repository sur GitHub
# Nom: tradesignal-pro
```

### 2. Cloner et ajouter les fichiers

```bash
git clone https://github.com/votre-username/tradesignal-pro.git
cd tradesignal-pro
cp /chemin/vers/tradesignal_pro.html index.html
git add index.html
git commit -m "Initial commit: TradeSignal Pro v1.0"
git push origin main
```

### 3. Activer GitHub Pages

1. Allez sur votre repository GitHub
2. Cliquez sur **Settings** → **Pages**
3. Source: Deploy from a branch
4. Branch: **main** / **root**
5. Cliquez **Save**
6. Attendez 2-3 minutes
7. Votre app est live sur: `https://votre-username.github.io/tradesignal-pro/`

## 📱 Utilisation

1. **Sélectionnez une paire** - BTC/USDT, ETH/USDT, etc.
2. **Choisissez un timeframe** - 1s, 3s, 5s, 15s, 30s, 1m, 3m, 5m, 15m, 30m, 45m, 1h
3. **Activez les alertes sonores** - Cliquez sur l'icône 🔊
4. **Attendez les signaux** - L'IA analyse et génère des signaux BUY/SELL
5. **Consultez l'analyse** - Section "Analyse IA" pour les explications

## 🔧 Configuration

### Intervalle de rafraîchissement
- 1 seconde - Ultra rapide (scalping)
- 3 secondes - Rapide (par défaut)
- 5 secondes - Standard
- 10 secondes - Économie de données

### Types de graphiques
- Ligne - Vue simplifiée
- Bougies - Vue détaillée (OHLC)

**TradeSignal Pro est un outil éducatif.** Les signaux générés par l'IA sont basés sur des algorithmes techniques et ne constituent pas des conseils financiers. Tradez toujours à vos risques et faites vos propres recherches.
